- Una standard ACL solo mira la `IP address origin/source`
- Una extended ACL revisa las `IP src/dest` y ademas los `puertos src/dest (TCP/UDP)`
- Tambien puede verificar por el tipo de IP (ICMP, RIP, OSPF, IPSEC, etc).
